## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- GUI with modern liquid glass design
- Live preview for DV cameras
- DV capture with ffmpeg
- Automatic merging of multiple capture parts
- AI-based upscaling with Real-ESRGAN
- Plex export in standard format
- Cover generation with Stable Diffusion
- Multiple upscaling profiles (HQ, Balanced, Fast)
- Configurable settings via JSON
- Comprehensive logging system
- PyInstaller build support
- Download manager for missing components
- Cross-platform support (Linux, Windows, macOS)
